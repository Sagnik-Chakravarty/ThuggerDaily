# src

Reusable dashboard and analysis modules.

## Key Modules

- `load_data.py`: cached CSV loading and schema validation.
- `preprocessing.py`: standardization helpers.
- `engagement.py`: engagement counts and engagement-rate calculations.
- `sentiment.py`: sentiment aggregation and smoothing.
- `topic_modeling.py`: topic prevalence and shift utilities.
- `topic_leveling.py`: seven-topic to multi-level interpretation mapping.
- `statistics.py`: descriptive summaries, tests, effect sizes, lag correlations, and regressions.
- `causal.py`: event windows, event study, DiD, ITS, and ThuggerDaily-aligned attribution estimates.
- `llm_reporting.py`: optional local Ollama report generation with a template fallback.
- `plotting.py`: Plotly visualizations.
- `ui_components.py`: shared Streamlit UI helpers and theme styling.
