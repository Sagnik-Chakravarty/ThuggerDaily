# Codes

Original project notebooks and scripts for scraping, data collection, cleaning, and sentiment analysis.

## Contents

- `Data Collection/`: scraping/API notebooks and scripts for public web, social, YouTube, Spotify, and related sources.
- `Data Cleaning/`: transformation notebooks used to prepare source-specific datasets.
- `Sentiment Analysis/`: earlier analysis notebooks for sentiment scoring and visualization.

These files document the research pipeline that feeds the `Data/` folders and, indirectly, the Streamlit dashboard ingestion script.
